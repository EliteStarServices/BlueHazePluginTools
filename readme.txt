* BH / ESS Plugin API and Tool

== Description ==

= Quickly Find the Plugin You're Looking For on the Plugins Screen =

Start typing and the list of plugins will be filtered showing only the plugin names that match your search.

= Output Basic Plugin Information to Dashboard and as JSON =

Output as Enhanced Plugin Info, JSON TextArea and as JSON File / API

= Install Plugins using the Download URL =

Skip the download, then upload method and install plugins directly from a download URL

= Easily Download Plugins & Themes from the Dashboard =

Add direct download ability to Installed Plugins and Themes pages

= Show Plugin Condition / Last Updated Info on Installed Plugins Page and as a Dashboard Alert =

Get Info about possibly abandoned, 3rd Party or Unofficial plugins on the dashboard and the plugins page

= See Which Plugins Are Active on Each Site in a Multisite Network if Network Activated =

The Plugins Report feature located under the Network Admin Plugins section will show which sites a plugin is active on. It will also give you a list of plugins that aren't activated on any site.
